# US-07-05 — Interaction

| Action | Control | Result |
|--------|---------|--------|
| Tap + track row | Virtual add-track row | `addTrack` |
| Long-press track | Track header or lane | In-place popup: Add MIDI / Add Audio Clip |
| Add MIDI Clip | Popup menu | Clip at press beat (grid, non-overlapping) |
| Add Audio Clip | Popup menu | Library sheet → clip at press beat |
| Save / Open project | Settings tab | System archive dialog |
