# US-07-05 — Interaction

| Action | Control | Result |
|--------|---------|--------|
| Tap + track row | Virtual add-track row | `addTrack` |
| Long-press track | Track header or lane | Clip context menu |
| Add MIDI Clip | Context menu | `createMidiClip` on that track |
| Add Audio Clip | Context menu | Sample picker → `createSampleClip` |
| Save / Open project | Settings tab | System archive dialog |
