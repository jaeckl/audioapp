# US-10-03: 4×4 MPC pad grid

## User story

As a **user** on a sampler track, I see a **4×4 pad grid** with flash feedback and Y-position velocity.

## Acceptance criteria

- [x] 16 pads, min 48dp touch targets
- [x] Pad flash on trigger (~100ms)
- [x] Held state visual while finger down
- [x] Y-position → velocity (bottom=loud default)
- [x] Bank tabs A/B/C/D (pitches offset by 16 semitones)
- [x] Auto-select Pads for sampler tracks
