# US-10-04: Play keyboard (chromatic + In Key)

## User story

As a **user** on an instrument track, I can play a **compact keyboard** with octave shift, scroll mode, and scale-locked note bars.

## Acceptance criteria

- [x] Chromatic mini keyboard (~14 keys visible)
- [x] Octave ± with indicator
- [x] Glissando / Scroll toggle (scroll moves key window)
- [x] In Key mode: equal note bars, wrong notes hidden
- [x] Key + scale picker (major/minor/pentatonic)
- [x] Multi-touch polyphony (3+ fingers)
- [x] Y-position velocity on strike
- [x] Auto-select Keys for oscillator tracks
