# US-11-03-ux-ui: Dual oscillators + unison — UX & UI

## Type

UX / UI

## Parent feature

[US-11-03](US-11-03-dual-oscillators-unison.md)

## Design intent

Osc2 and unison controls feel like **one shared voice engine** — grouped under osc section, not a second instrument.

## Layout & hierarchy

- Osc2 level slider or knob adjacent to osc1 level (temporary layout until Osc tab)
- Unison: voice count stepper (1–4) + detune knob

## Visual states

| State | Treatment |
|-------|-----------|
| Unison = 1 | Detune knob dimmed |
| Osc2 level = 0 | Osc2 params still editable |

## Acceptance criteria (visual)

- [ ] Unison cap (4) visible in UI copy or max on control
