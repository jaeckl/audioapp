## Type

UX / UI

## Parent feature

## Design intent

Four-tab synth matches sampler ADSR/filter familiarity — **Filter** tab shows “LP12” badge so users know multimode is not available yet.

## Layout & hierarchy

- Header tabs: Osc | Mix | Filter | Amp (flat, dark rounded-top selection per device_header_tab_bar)
- Filter: cutoff + resonance knobs, ADSR row, env amount
- Amp: ADSR only (no duplicate gain — level panel)

## Copy & feedback

- [ ] ADSR labels A/D/S/R consistent with sampler
