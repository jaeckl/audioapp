## Type

UX / UI

## Design intent

Mix mode selector reads as **oscillator algebra** — compact segmented control or dropdown with short labels: Mix, Neg, AM, Sign, Max.

## Layout & hierarchy

- Noise level knob
- Mix mode control between osc level knobs

## Copy & feedback

- Tooltips or subtitles for cryptic modes (AM = ring mod style)

## Acceptance criteria (visual)

- [ ] Active mix mode clearly indicated
- [ ] All five modes reachable without scrolling on phone landscape strip
